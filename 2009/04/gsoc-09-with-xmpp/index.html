The XMPP Blog

GSoC '09, with XMPP

The XMPP Standards Foundation has helped support a number of open-source projects through Google's Summer of Code program over the years. Although the XSF is not participating in the Google Summer of Code this year, a number of XMPP-related projects have been accepted by other mentoring organizations...

On the JabberFr forums, "Misc" posted the full list:

  • Pidgin:
    • XMPP - libpurple transport, allowing servers to offer gateways to all the IM networks that Pidgin (or Adium) supports
    • Privacy Rewrite, giving users powerful controls over "XMPP firewalling"
    • Telepathy prpl, integrating Pidgin's powerful Purple library with the GNOME Instant Messaging and Presence framework
  • Apache Foundation:
    • Extend the Vysper XMPP Server/Client with the publish-subscribe XEP (XEP-060)
  • Python Foundation:
    • Gajim: Whiteboard Implementation
    • Adding support for server-side history and roster versioning in Gajim
  • Gnome :
    • GNOME-Games + Telepathy: Play GNOME-Sudoku with your IM contacts
    • Implement Music Sharing in Banshee using the Telepathy API
  • KDE :
    • VoIP client for KDE (revive KCall)
    • KDE - Conversation Logging Framework
    • Turn Kopete chat window into a library
  • Maemo :
    • IM Client For Canola Using Python-purple

At the XSF, we're tremendously excited to see so much implementation of cutting edge technologies in so many widespread projects. It's a testement to the innovative nature of the XMPP community, and we're looking forward to seeing - and using - the results.

Article by Misc, translated by Nicolas Vérité, edited by Dave Cridland.
